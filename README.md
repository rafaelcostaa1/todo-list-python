# To-Do List em Python

Projeto simples de lista de tarefas desenvolvido em Python, com armazenamento em arquivo JSON.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias

- Python
- JSON

## Como executar

1. Clone o repositório:
git clone https://github.com/seu-usuario/todo-list.git

2. Acesse a pasta do projeto:
cd todo-list

3. Execute o programa:
python main.py

## Observação

As tarefas são armazenadas automaticamente no arquivo tarefas.json.
